Express

Express is one of the most popular Node.js modules allowing to easily expose server side resources to HTTP clients such as browsers and as APIs.

Install Express

Use npm to install express as follows:

> npm install express

Create and Start a Server Instance

Using express it is very simple to create and start a server. Simply load the module with require(), create an instance of the module, and call listen() as shown in the sample file server.js shown below: 

var express = require('express');
var app = express();

app.listen(3000);

To start the server, excute the server.js file with node.js from the command line as follows: 

> node server.js

Respond to HTTP Requests

Express allows mapping HTTP request methods and URLs to backend code. Use functions such as get(), post(), and put() to respond to HTTP requests. These functions accept as parameters URL patterns and function. When an incoming URL matches the URL pattern, the function is invoked. Below is an example of URL /hello that is mapped to a function that returns Hello World from server.js 

var express = require('express');
var app = express();

app.get('/hello', function(req, res){
    res.send('Hello World from server.js!');
});

app.listen(3000);
