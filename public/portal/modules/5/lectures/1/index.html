Introduction to Mongoose

OVERVIEW: In the previous section we used the mongo client to interact with a MongoDB instance. In this section we will use the mongoose library to interact with MongoDB programmatically from our Node.js applications. First we install the mongoose library for MongoDB. Then we create schemas and models and expose them as REST Web services. Finally we map REST URL patterns to invoke HTTP services that create, read, update, and delete (CRUD) instances of the Mongoose models.
