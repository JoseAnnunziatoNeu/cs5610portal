Introduction to MongoDB

OVERVIEW: In this section we will install and configure MongoDB, and use the mongo client to connect and interact with a MongoDB instance. We will use the client to show how to insert, search, update and delete JSON data to the database. In later sections we will show how to accomplish the same programmatically from a Node.js application, while making the data through a REST Web service.
