Lecture 2: CSS

OVERVIEW: CSS, or Cascading Stylesheets, is a simple language used to format the look and feel of a Web page. Whereas HTML formats the overall structure and logical organization of a Web page, CSS formats how the page looks. CSS formats the font to use, the colors of text, background, and borders, the white spaces between various elements, and page layout.
