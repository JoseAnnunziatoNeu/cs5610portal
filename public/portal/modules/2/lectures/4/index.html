Lecture 5: Bootstrap

OVERVIEW: The Web development industry has matured enough for design patterns and best practices to emerge. There are enough collective best practices that they can be grouped into coherent frameworks. Several of these frameworks have emerged and become fairly popular amongst the developer community. Twitter's Bootstrap is one of the most popular CSS frameworks that allow developers to quickly style and layout a Web page into consistent, professional looking pages applying industry statndard, best practices.
