Box Model

OVERVIEW: The content in a Web page basically consists of various rentangular regions that can render with the content flow, render vertically or horizontally, render with a specific width, height, border, and white space based on various rules and sizes of the content. Collectively these rules are captured as a "box model". The Box Model lecture covers the various properties that govern the box model.
